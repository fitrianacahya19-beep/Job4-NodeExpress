## Deskripsi Project
Project ini merupakan aplikasi web sederhana menggunakan Node.js dan Express sebagai server.

## Teknologi yang Digunakan
- HTML
- JavaScript
- Node.js
- Express.js

## Cara Menjalankan
1. Buka folder project.
2. Install dependency:
   ```bash
   npm install
   ```
3. Jalankan server:
   ```bash
   node server.js
   ```
4. Buka browser dan akses:
   ```
   http://localhost:3000
   ```
